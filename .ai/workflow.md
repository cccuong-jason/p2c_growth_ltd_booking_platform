# AI Workflow

Use this project as a small delivery branch:

1. Read `docs/P2C_Growth_PRD_v1.md` and `docs/P2C_Growth_UI_UX_Guidelines_v1.md`.
2. Keep changes scoped to Phase 1.
3. Run `npm run test`, `npm run lint`, and `npm run build` before delivery.
4. Document credential-dependent behavior instead of hardcoding secrets.
